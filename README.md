# 🌱 Smart Greenhouse Monitoring System

A microcontroller-based greenhouse system for monitoring environmental conditions in real time using an ESP32 and an LCD display.


## 📸 Project Overview

![Setup](setup.jpg)
---

## ⚙️ Features

* Temperature and humidity measurement (SHT21)
* Soil moisture monitoring (analog sensor)
* Real-time data display on LCD (I2C)
* Button-controlled LCD backlight

---

## 🧰 Hardware

* ESP32
* SHT21 sensor
* Soil moisture sensor
* I2C LCD display
* button
* 3D printet case

---

## 🚀 Setup

1. Clone the repository
2. Install required libraries:

   * `Wire`
   * `LiquidCrystal_I2C`
   * `SHT21`
3. Upload the code to your ESP32
4. Open Serial Monitor (115200 baud)
